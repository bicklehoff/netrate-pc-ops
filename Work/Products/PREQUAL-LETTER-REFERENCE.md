# Pre-Qualification Letter — Competitor Reference

**Source:** Bayou Mortgage (email + PDF templates, screenshots saved 2026-03-11)
**Purpose:** Reference for designing NetRate's borrower milestone communications and prequal letters

---

## Page 1: Milestone Email (Initial Approval Notification)

### Template Structure

### Email Preview Text
> Great news! Your loan has received initial approval. There are just a few remaining items we need to wrap up to get you to the closing table.

### Header
- Company logo (top center)
- Dark blue hero banner with celebration emoji
- Label: "LOAN UPDATE"
- Headline: **"Initial Approval Reached!"**
- Borrower name(s)
- Subject property address

### Body Copy
- Greeting: "Dear [All Borrowers First Names],"
- Explains what "Initial Approval" (Conditional Approval) means
- Sets expectations: a few final "conditions" remain
  - Documents from borrower (paystub, bank statement)
  - Third-party items (appraisal, homeowner's insurance)
- Reassurance: "Our team will be reaching out shortly"
- CTA: Link to POS portal to review status and outstanding conditions

### Loan Progress Tracker (Visual Timeline)
Vertical stepper showing completed and pending milestones:

| Step | Status | Date Field |
|------|--------|------------|
| Application | Complete (green check) | Loan App Intake Date |
| File Review | Complete | Loan Qualification Date |
| Pre-Approved | Complete | Loan Pre-Approved Date |
| Processing | Complete | Loan Setup Date |
| Disclosures Sent | Complete | Loan Disclosed Date |
| Submitted to Underwriting | Complete | Loan Submitted to UW Date |
| **Initial Approval** | **Current (highlighted, "YOU ARE HERE")** | **Loan Approved w/ Conditions Date** |
| Clear to Close | Pending (gray) | Pending |

### Footer
- Email signature

---

## Design Notes

- **Color scheme:** Dark navy blue header, white body, teal/green accents for completed steps
- **Current step highlight:** Light blue/teal background box with "YOU ARE HERE" label in green
- **Completed steps:** Green circle with white checkmark
- **Pending steps:** Gray circle with question mark
- **Dates:** Shown in teal/green text aligned right of each milestone
- **Overall tone:** Celebratory but informative — explains what's next without overwhelming

## What to Adopt for NetRate

- The visual loan progress tracker is excellent — borrowers can see exactly where they are
- Dynamic date fields per milestone (pull from LOS/POS data)
- "YOU ARE HERE" highlighting is a strong UX pattern
- Explaining conditions in plain language (not jargon) builds trust
- Portal CTA to check status self-serve reduces inbound calls
- Consider adding: specific conditions list, loan officer contact, estimated timeline to close

## Page 2: Pre-Qualification Letter (PDF Document)

### Header
- Company logo (top left)
- Green banner: "[Org Display Name] Pre-Qualification Letter"

### Borrower Info Block (2-column table)
| Left | Right |
|------|-------|
| **Borrower(s):** All Borrowers Full Names | **Date:** Quote Preparation Date |
| **Address:** Subject Property Address | **Expires On:** Pre-Approval Expiry Date |
| | **Ref #:** Quote Ref Number |

### Body
- Headline: **"CONGRATULATIONS!"**
- Standard prequal language: borrower has been pre-approved based on income, employment, credit documentation provided to [Org Name]
- Disclaimer: financing subject to final underwriting approval, clear & marketable title, homeowner's insurance, appraisal review, and verification that financial condition/creditworthiness has not changed between letter date and closing/funding

### Pre-Approval Terms (2-column highlight boxes)

| PURCHASE PRICE | MAX LOAN AMOUNT |
|----------------|-----------------|
| Scenario Property Value | Scenario Base Loan Amount |

### Loan Details (2-column layout)
**Left column — Loan Terms:**
| Field | Value |
|-------|-------|
| Loan-to-Value | Scenario LTV |
| Loan Term | Scenario Loan Term |
| Loan Type | Scenario Mortgage Type |

**Right column — Verification Checklist:**
| Item | Status |
|------|--------|
| Credit Reviewed | Green checkmark |
| Income Documented | Green checkmark |
| Assets Verified | Green checkmark |
| AUS Approval | Green checkmark |

### Legal Disclaimer
> This is not a commitment to lend; nor is it a commitment of interest rate or fees. Changes to income, employment, assets, or credit profile may render this pre-qualification null and void.

### Signature Block
- "Sincerely,"
- PDF Signature
- **Loan Officer Full Name**
- Phone: [Loan Officer Phone]
- Email: [Loan Officer Email]
- Org Display Name (right-aligned)
- NMLS: [Loan Officer Company NMLS] (right-aligned)

## Design Notes (PDF Letter)

- **Clean, professional layout** — white background, minimal color usage
- **Green accents** on verification checkmarks and header banner
- **Two hero boxes** for purchase price and loan amount — large bold text, high visual weight
- **Verification checklist** with green checkmarks adds credibility — shows the borrower this isn't just a guess
- **Expiration date** prominently displayed — creates urgency for realtors/sellers
- **Ref number** for tracking — ties back to POS/LOS system

## What to Adopt for NetRate (Both Templates)

### From the Email (Page 1)
- Visual loan progress tracker — borrowers see exactly where they are
- Dynamic date fields per milestone (pull from LOS/POS data)
- "YOU ARE HERE" highlighting is a strong UX pattern
- Plain language condition explanations build trust
- Portal CTA for self-serve status checks reduces inbound calls

### From the PDF Letter (Page 2)
- Verification checklist (Credit, Income, Assets, AUS) — differentiates a real prequal from a basic prequalification
- Purchase price + loan amount as hero numbers — this is what realtors/sellers care about
- Expiration date — standard but essential
- LTV, term, loan type summary — concise loan parameters
- Clean signature block with NMLS — compliance covered
- Consider adding: rate lock status, estimated monthly payment, specific program name
